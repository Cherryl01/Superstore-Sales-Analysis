
📊 Superstore Sales Analysis (Python Project)
📌 Project Overview

This project is a data analysis of Superstore sales using Python. The goal is to explore sales performance, profitability, customer behavior, and regional trends using data visualization and exploratory data analysis (EDA).

The insights generated help in understanding business performance and making data-driven decisions.

🎯 Objectives
Analyze overall sales and profit performance
Identify top-performing product categories
Explore regional sales distribution
Analyze monthly sales trends
Understand customer segment contributions
Identify top-selling products
Examine relationships between key variables (sales, profit, discount, quantity)

📂 Dataset
Source: Kaggle Superstore Dataset
Link: https://www.kaggle.com/datasets/vivek468/superstore-dataset-final

🛠 Tools & Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Google Colab

📊 Key Analysis Performed
1. Data Cleaning
Removed duplicates
Converted date columns to datetime format
Created new time-based features (Month-Year)
2. Exploratory Data Analysis (EDA)
Sales distribution analysis
Profit analysis by category and region
Customer segment analysis
3. Visualizations
Bar charts for category and region performance
Line charts for monthly sales trends
Pie charts for customer segments
Heatmap for correlation analysis

❓ Research Questions Answered
Which product categories generate the highest sales?
Which product categories generate the highest profit?
Which region performs best in terms of sales?
What are the monthly sales trends over time?
Which customer segment contributes the most to sales?
What are the top 10 products by sales revenue?
What relationships exist between sales, profit, discount, and quantity?

📈 Key Insights
Technology and Office Supplies are strong contributors to revenue
Some regions consistently outperform others in sales
Certain products dominate overall sales performance
Sales trends show seasonal variation over time
Profitability is influenced by discount levels and product category
